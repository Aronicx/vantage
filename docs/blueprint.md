# **App Name**: Vantage Point

## Core Features:

- Island Synthesis Engine: Generate a unique procedural island landmass with complex topographies and randomized natural biomes.
- Geopolitical AI Lore Tool: Utilizes a reasoning tool to synthesize unique historical narratives, diplomatic relationships, and naming conventions for every nation generated.
- Dynamic Heraldry Designer: Randomized flag generation based on color theory and national history synthesized during the world-building phase.
- Interactive Tactical Cartography: High-fidelity map interface serving as the main command center, featuring togglable overlays for borders, military reach, and economic flow.
- Sovereign Settlement Hierarchy: Automated placement of capital cities, minor townships, and strategic military outposts across the world map based on geographic defensibility.
- World Persistence Engine: Database management system to store complex world states, enabling long-term offline play sessions for a single-player sandbox experience.

## Style Guidelines:

- The palette is anchored in a concept of 'Diplomatic Intelligence'. The primary color is a refined Command Blue (#326194) used for military highlights.
- The background is a deep, desaturated Midnight Navy (#0F1216) to provide a high-contrast foundation for glowing vector map lines.
- The accent color is a sharp Electric Glaze (#3DBCCF), used for interactive tactical borders and cursor-sensitive map markers.
- Font pairing: 'Space Grotesk' for technical data displays and headers, and 'Inter' for UI labels and lore text, ensuring high legibility in dense data views.
- Minimalist vector symbols for settlements and outposts, inspired by mid-century nautical charts and modern intelligence HUDs.
- Full-screen edge-to-edge canvas with floating sidebar modules that provide contextual information about hovered countries and cities.
- Smooth pathfinding animations and subtle territorial pulse effects to signify shifting border influences or active military presence.
